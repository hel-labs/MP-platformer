# Features
## Mario inspired sidescroller/platformer overworld
Features a fully visual sidescrolling overworld similar to classic Mario. Wandering enemies throughout the levels, hazards, breakable objects, collectibles and basic interactions. To be polished further.
## Pokemon inspired turn-based battle system
Features a fully functional battle system with a unique blend of turn-based combat, undertale-inspired conversation system, and dice-roll based damage strategy.
## Undertale inspired conversation system 
Features a work-in-progress conversation system within battles. Favourable dialogues lower hostility and damage, unfavourable dialogues increase hostility and damage greatly, and neutral dialogues do nothing.
Boss battles on each level expand upon this system: an initial phase with only a series of riddles, answering them correctly granting access to the actual battle where further conversations can be made or the enemy can be fought.
## Persistent local leaderboard
Features a persistent record of scores of all runs. Records are sorted by username; with points and progress contributing to overall points.
## Minimal unique ending based on playstyle (Agrressive, Neutral, Passive)
Features a minimal ending system where the final dialogues will be different based on playstyle:
1. Aggressive (Much more enemy killed than spared)
2. Neutral (Almost similar amount of enemy killed and spared)
3. Passive (Much more enemy spared than killed)
## Inventory and Point System
Features a work-in-progress basic inventory system that affects overworld interaction(tile breaking, movement speed etc) and battle strategy (Status mutations); alog with a functional point system based on enemies killed/spared, coins collected and levels cleared.
